# ntdomainsearch

Ön yüz domain arama modülü.

## Sorumluluklar

- Ana sayfaya eklenebilir domain arama kutusu
- Ajax ile domain uygunluk sorgulama
- ResellerClub `domaincheck.httpapi.com` endpoint kullanımı
- Uygun domain için sepete ekleme akışı
- Ana `ntresellerclub` modülünden API ayarlarını okuma

Bu modül, ana otomasyon modülünden ayrı geliştirilecektir.
