# ntresellerclub

Ana PrestaShop modülü.

## Sorumluluklar

- API ayarları
- Lisans doğrulama
- ResellerClub API client
- Customer / contact / order provisioning
- Hizmet takip tabloları
- Cron ve yenileme mailleri
- Müşteri Hizmetlerim paneli

İlk çalışan PrestaShop modül iskeleti bu klasörde oluşturulacaktır.
