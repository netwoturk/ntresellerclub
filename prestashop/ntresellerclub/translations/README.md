# Çoklu Dil Yapısı

Bu klasör modül içi çeviri planını tutar.

Desteklenen diller:

- tr: Türkçe
- en: English
- de: Deutsch
- es: Español
- fr: Français
- it: Italiano

PrestaShop 1.7/8/9 için modül metinleri `l()` ve Smarty `{l}` yapılarıyla çevrilebilir kalacaktır. Mail şablonları `mails/{lang}/` klasörlerinden yönetilir.
